Instructions pour jouer à War of Futur

Comment exécuter le jeu

  1. Téléchargez le fichier wof.sb3 en cliquant sur le bouton "Obtenir le jeu".
  2. Ouvrez le fichier téléchargé avec un logiciel compatible, comme Scratch.
  3. Appuyez sur le bouton Run pour commencer à jouer !

Créer un raccourci avec l'icône

  1. Localisez le fichier wof.sb3 dans votre dossier de téléchargement.
  2. Cliquez avec le bouton droit sur le fichier et sélectionnez Créer un raccourci.
  3. Déplacez le raccourci sur votre bureau pour un accès facile.
  4. Pour changer l'icône, cliquez avec le bouton droit sur le raccourci, choisissez Propriétés, puis Changer d'icône.
  5. Sélectionnez l'icône que vous souhaitez utiliser et cliquez sur OK.
